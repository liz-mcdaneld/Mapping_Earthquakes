# Mapping Earthquakes

## Overview

The purpose of this project it to create a map using JavaScript and APIs to show the latest Earthquake data. API requests are made to a server to host geographical maps.  Using JavaScript and the Data-Driven Documents (d3) library to populate geographical maps with GeoJSON data. Multiple map layers are added using Leaflet control plugins. JavaScript ES6 functions are used to add GeoJSON data, features, and interactivity to maps. Render maps on a local server.
